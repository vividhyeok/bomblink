# Fuse Link Prototype

2000년대 삼성 애니콜 피처폰 게임 **Bomb Link**에서 영감을 받아 만든 비공식 웹 리메이크입니다.

원본 ROM, 그래픽, 사운드, 리소스를 사용하지 않고, Bomb Link의 플레이 감각을 참고해 새로 구현한 프로토타입입니다.

## 실행 방법

```bash
npm install
npm run dev
````

## 조작법

| 키             | 동작       |
| ------------- | -------- |
| 방향키 / WASD    | 커서 이동    |
| Space / Enter | 폭탄 심지 회전 |
| Z / Backspace | 반대 방향 회전 |
| M             | 음소거      |
| R             | 재시작      |
| Esc           | 일시정지     |

## 게임 방식

폭탄의 심지 방향을 돌려 서로 연결하고, 불꽃이 심지에 닿으면 연결된 폭탄들이 연쇄적으로 폭발합니다.

많은 폭탄을 한 번에 연결해 터뜨릴수록 높은 점수를 얻을 수 있습니다.

## 프로젝트 성격

이 프로젝트는 **Bomb Link의 플레이 감각을 재현하기 위한 개인 학습용 리메이크**입니다.

상업적 목적이 없으며, 원본 자산을 포함하지 않습니다.
